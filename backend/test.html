Benvenuto all'API di FakeRestaurant!

Questa è un'interfaccia di test per verificare il funzionamento degli endpoint del server. Puoi usare strumenti come Postman o curl per testare le API in modo più approfondito.

Endpoint Disponibili

GET /api/health - Controlla lo stato del server.
GET /api/restaurants - Ottieni la lista dei ristoranti.
GET /api/restaurants/:id - Ottieni un ristorante specifico.
POST /api/auth/register - Registra un nuovo utente.
